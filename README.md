## ✨ What is this?

A clean, fast personal website and portfolio built for Nishan Patel. Designed to showcase his work, skills, and background in a professional, modern layout. Built for performance and SEO from the ground up.

## 🛠 Tech Stack

| Layer | Technology |
|---|---|
| Framework | Next.js |
| Language | TypeScript |
| Styling | Tailwind CSS |

## 🚀 Getting Started

```bash
git clone https://github.com/tsvtnv/NishanPatel.git
cd NishanPatel
npm install
npm run dev
```
